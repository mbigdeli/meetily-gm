Miting — Product Plan

AI meeting minutes. Local, private, free — with the integrations everyone else charges for.
Planning package · July 2026 · Mohamad Bigdeli · repo: meetily-gm → miting · site: miting.bigde.li
Every commercial meeting-notes tool paywalls its integration layer (Granola $14+, tl;dv up to $59/user/mo). Miting gives that layer away: on-device Whisper transcription, AI speaker diarization, summaries through the user's own LLM (Codex CLI / Ollama / any key), push to Jira & Slack, a local MCP server for Claude & ChatGPT — and first-class فارسی support no competitor offers. Open source, no cloud backend, meetings never leave the machine.
